The main tab of EcTracker has four components

  1. Home: main home page of the server.
  2. Analysis: all the analysis of the single-cell dataset will be done under this tab
  3. Tutorial: video showing how users can use our server and also documentation.
  4. Contact Us: information about developers, web designers, testers, and collaborators

  1. The analysis tab has multiple sub-tabs with the Upload tab as first.

In Upload tab, the user can:

  1. Upload a CSV file (comma separated file) in Upload expression matrix
  2. If the user has 10X genomics dataset files, they can upload multiple files i.e. barcode.tsv, genes.tsv, and matrix.mtx in Upload 10X data files
  3. Sample dataset which is a CSV file (comma separated file)

The file upload may take some time depending on the size of the dataset.

After uploading the expression matrix, the user can:

  1. See a snippet of their dataset
  2. See the dimension of their dataset

After uploading the data, the user will go to scData Processing tab for further analysis. The user must click the required action button “One-Click Analysis” to start the analysis.

After clicking the action button, a popup will appear notifying that the process has started running and will take some time to run the processes depending upon the user’s dataset.

In the scData Processing tab, basic single-cell analysis will be following the Seurat pipeline. The following results will be obtained:

  1. Variable feature plot
  2. Variable features
  3. PCA plot
  4. Heatmap plot
  5. Jackstraw plot
  6. Elbow plot
  7. UMAP

The user can download the plots using the Download Plot button provided below each plot. We have also provided explanations of the plots.

There are “Top” buttons at the bottom of all the pages using which the users can move to the top of the page.

DEG Analysis tab performs differential gene expression analysis. Here:

  1. The slider widget allows the user to choose the log fold change threshold which is 1 by default.
  2. There is a Download link available to download the differentially expressed genes

The users also have the option to upload the meta data file (in CSV format) after which they have to click on “Table with Meta File” to generate meta-file table. This metadata file should contain a column that matches the cell type and this column should have a column heading as “Samples”. A Cluster identity table will be provided if no meta data file is uploaded by the user.

In CellEnrich tab:

  1. There are two methods: AUCell and Stouffer Score
  2. In AUCell, the users have the option to select different signatures for cell enrichment analysis. AUCell gives a cell ranking plot, AUC curve with cell assignments according to the selected signature and UMAP of the selected cells.

In Stouffer score tab, the user can select the Signatures for which the Stouffer score and one-sided Wilcoxon P-value will be calculated.

TissueEnrich tab has two methods:

  1. Hypergeometric method and Stouffer score method
  2. In Hypergeometric test, the user can select the cluster for which they want to perform the analysis. They can also download the selected cluster table.
  3. The user can choose between two datasets i.e. HPA and GTEx for which tissue enrichment will be performed.

The user can select the tissue and the respective transcripts for which they can see the UMAP and genomic location.

The second method in TissueEnrich is Stouffer Score, where there are different tabs for HPA and GTEx datasets. Both have their respective signature which will be used for Stouffer score calculation.

In Gene Regulatory Network tab:

  1. There are three subsections:Cell Cluster, CellEnrich, TissueEnrich
  2. In Cell Cluster, the user can select the cluster for which gene regulatory network will be generated.
  3. CellEnrich tab gives regulatory network for signatures.
  4. TissueEnrich tab gives networks for both HPA and GTEx dataset signatures.

In both CellEnrich and TissueEnrich tabs, two network graphs will be displayed with links between target genes + signature and transcription factors + target genes. A table is also

provided for the same.

Note: Please allow all the default processes to run and load by visiting all the tabs before changing any of the parameters such as changing the threshold, and selecting the clusters.
